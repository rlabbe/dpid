# dpid
Fast Python implementation of Rapid, Detail-Preserving Image Downscaling paper by Weber, et. al.

Used numba to get extremely fast image generation.

this is not carefully tested, I wanted to see how it worked with sheet music, results are good
but too blocky for my needs. Claude wrote most of this, not me.
